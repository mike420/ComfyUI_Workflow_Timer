# ComfyUI Workflow Timer

<p>A simple workflow timer node for ComfyUI.</p>

<img width="964" alt="workflow timer node" src="https://github.com/mike420/ComfyUI_Workflow_Timer/blob/main/img/workflow_timer.png">

## Installation

### Comfy CLI (recommended)

```bash
comfy node install ComfyUI_Workflow_Timer
```

You can also find it on the [ComfyUI Registry](https://registry.comfy.org/publishers/mike420/nodes/ComfyUI_Workflow_Timer).

### ComfyUI Manager

Search for `ComfyUI Workflow Timer` in the Custom Nodes section and install it from there.

### Manual

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/mike420/ComfyUI_Workflow_Timer.git
```

Restart ComfyUI and refresh your browser. That's it.
